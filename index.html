Overview

This is the documentation for the Opportunity Sharing ("OppShare") Program.

OppShare automates the delivery of shared opportunities to Slack and email groups. The opportunities are initially shared by individuals through a Google Form, and they are stored in a Google Sheet.

Details

OppShare has the following deliverables:

Slack digest
Delivers - to a specific Slack channel - all opportunities that have been shared within a time range (ex: the last 14 days).

Slack single opportunity posts
Creates a Slack post each time an opportunity has been shared through the Google Form.

Email digest
Delivers via email all opportunities that have been shared within a time range (ex: the last 14 days).

Technical Details

OppShare is a Google Apps Script project, which is written in the Google Apps Script (GAS) language, which is essentially javascript.

Time-based triggers are used to trigger the Slack and Email digests. An "From spreadsheet - On form submit" trigger is used for the Slack single opportunity posts.

Understanding the files

EmailDigest.html
Dynamic HTML tempalate for email digest

EmailOpportunities.js
Script for Email digest. The "emailOpportunities" function is triggered by a time-based trigger.

Helper.js
Contains dictionaries of variables which are used throughout the project.

SlackOpportunities.js
Script for Slack digest and single opportunity posts. The "slackOpportunities" function, which creates the Slack digest, is triggered by a time-based trigger. The "onFormSubmit" function, which creates the single opportunity posts, is triggered by a "From spreadsheet - On form submit" trigger.
